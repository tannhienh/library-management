## Application console Library Management

Compiler: GNU C++, MSVC

### USAGE

- **Build:**    
> `$ make`
>
> or
>
> `$ make build`

- **Build Debug:**
> `$ make debug`

- **Run:**
> `$ ./main`

- __Remove files object *.o and execute file:__
> `$ make clean`

- **Remove data file:**
> `$ make clean-data`
